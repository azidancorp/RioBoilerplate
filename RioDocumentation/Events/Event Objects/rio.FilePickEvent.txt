FilePickEvent

Holds information regarding a file upload event.

This is a simple dataclass that stores useful information for when the user
picks a file using a [`FilePickerArea`](/docs/api/filepickerarea). You'll typically receive this as
argument in [`on_pick_file`](/docs/api/filepickerarea) events.

Attributes

file
`rio.FileInfo`

Handle to the uploaded file.
