# Rio Documentation Snapshot

This directory contains a local text snapshot of the official Rio documentation for `rio-ui` 0.12.

Source: https://rio.dev/docs
Fetched from the crawler-rendered Rio page state embedded in `globalThis.initialMessages`.
Verified by loading every manifest URL in Chromium with Playwright, including the
`/docs/tutorial-tic-tac-toe/1-rio-setup` alias for the Rio Setup page.

The files are grouped to mirror the current documentation sidebar: Get Started, Concepts, Tips and Tricks, Cookbook, Deployment, Components, Routing, and Events.
